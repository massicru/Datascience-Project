# Datascience-Project

## Dataset geografico

Il dataset utilizzato per la realizzazione della mappa geografica **non è incluso** in questo repository GitHub a causa delle sue dimensioni.

Per riprodurre completamente l'analisi, è possibile scaricare il dataset tramite il **link presente nelle slide della presentazione**.

Una volta scaricato:

1. Copiare il file nella cartella prevista dal notebook;
2. In alternativa, aggiornare il percorso del file nella cella dedicata al caricamento del dataset.

> **Nota:** anche in assenza del dataset geografico esterno, il notebook può eseguire correttamente tutte le altre analisi e visualizzazioni che non dipendono dalla mappa geografica.
